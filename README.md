# Solmira Linux

Solmira Linux is a free and open-source Linux distribution based off Arch Linux. The goal of Solmira Linux is to provide an easy-to-use distribution with the base of Arch.

## Build

Clone the repository.

```bash
git clone https://github.com/Solmira-Linux/SolmiraLinux.git
```

Create the live image.
```bash
cd solmira
sudo mkarchiso -v .
```

The image will be created in the out folder.
